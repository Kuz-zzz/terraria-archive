TShock

    • Developer Pryaxis
    Status Up-to-Date
    Platform Windows, Linux, MacOS
    Tags

    Server

    TShock is a toolbox for Terraria servers and communities. That toolbox is jam packed with anti-cheat tools, server-side characters, groups, permissions, item bans and tons of commands. It's one of a kind.

    TShock supports 3-rd party plugins, which can be found in the TShock Plugins section of this website (soon, WIP).

    Back in 2013, before the 1.2 release, the Pryaxis team was sent the new Terraria version early to make TShock compatible with it on launch. However, a developer with a nickname Zidonuke leaked 1.2. This person contributed a lot to the development of TShock, mainly SSC, so Pryaxis tried defending him. This is when they lost Re-Logic's trust and thus Terraria's focus shifted away from public servers and closer to just playthroughs with friends.

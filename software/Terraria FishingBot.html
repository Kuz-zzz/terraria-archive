Terraria FishingBot

    • Developer GitMyCode
    Status Deprecated
    Platform Windows
    Tags

    Fishing

    A fishing bot for Terraria. Works only with default bobbers and only in a well lit area.

    Functionally identical to Terraria Auto Fisher Lite.

    Basically, don't use this bot.

    If you need a more reliable fishing bot and don't need to catch anything specific, use Izzy's Fishing Bot.

    If you need to catch specific things, use Terraria Fishing Bot.

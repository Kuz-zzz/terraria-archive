ZazaClient

    • Developer lucar901
    Status Up-to-Date
    Platform Windows
    Tags

    Cheat

    Client

    Zaza Client is a powerful cheat client for Terraria with a proper in-game UI. Starting off as a cheat table for Cheat Engine (which no longer works), it has now evolved into a full client.

    While it's not open source, it can be easily decompiled and compared to the base game with a code comparison tool.

    To open the Zaza menu, press "~".

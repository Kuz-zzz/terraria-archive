Terraria Fishing Bot

    • Developer Shim06
    Status Up-to-Date
    Platform Windows
    Tags

    Fishing

    Terraria Fishing Bot is a program that uses image recognition to automatically fish specific things. This Python script uses PyAutoGUI and OpenCV to scan the screen of any text that come from fishing while using a sonar potion.

    From my testing, this tool will catch any crate even if you only add a specific one to the catch list.
    Works better with a dark plain background, can skip catches otherwise.

    If you need a more reliable fishing bot and don't need to catch anything specific, use Izzy's Fishing Bot.

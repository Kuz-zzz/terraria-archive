Terraria Unified (.NET 8+, 64-bit, cross-platform server-client distribution) is a simple project that ports the game to .NET 8 and merges the Windows, macOS, and Linux client and server codebases into one. This modification maintains compatibility with vanilla servers, and tries its best to automatically detect existing Terraria installs in order to use the content folders of a preexisting installation.

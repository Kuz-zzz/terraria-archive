    World Editor

    TEdit is a free open source map editor for Terraria made by BinaryConstruct with major contributions from RussDev7 and other developers. The first version was released on May 24, 2011, for Terraria 1.0.2.
    TEdit also features extensive documentation with different tips and tutorials.

    Web editor was originally developed by cokolele/dzubyy. In January 2021 him and BinaryConstruct decided to collaborate on this project. Now it's fully maintained by BinaryConstruct.

    If you want to view the map in a browser, use TerraMap.

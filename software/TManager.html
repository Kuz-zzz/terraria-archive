TManager

    • Developer Jbro129
    Status Up-to-Date
    Platform Android
    Tags

    Player Editor

    World Viewer

    TManager is an inventory editor and map viewer with the ability to change some internal world values for Terraria on Android. It also has built-in player and world files submitted by other players available for download, a few useful seeds, a list of servers (most of which no longer work), news form the Terraria Forums and an Internet browser (just why?).

    Note that it has ads and will not work without Internet connection unless you pay $1.99.
